About Prodigy

Books and readers can be found anywhere. Reading made me aware of how difficult it is to locate books and get a copy that is readable. Prodigy was created as a one-stop store for this reason..

I created this project in a couple of weeks as my portfolio project to tie up the ALX School Software Engineering Program's end-of-foundations curriculum.
